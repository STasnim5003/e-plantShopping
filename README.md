# e-plantShopping
## Paradise Nursery Shopping Application

An interactive e-commerce web application for a plant shop called **Paradise Nursery**.  
Users can browse plants, add them to a cart, and view total costs dynamically.

## Features
- Display plants by categories
- Add to Cart functionality
- Cart quantity and total cost dynamically updated
- Continue shopping and checkout placeholders
